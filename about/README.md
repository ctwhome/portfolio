# About Page

A modern, smooth-scrolling portfolio/about page built with Next.js.

## Setup

1. Install dependencies:
   ```bash
   bun install
   ```

2. Run development server:
   ```bash
   bun dev
   ```

3. Build for production:
   ```bash
   bun run build
   ```

## Stack

- Next.js
- React
- Three.js
- GSAP
- Sass
